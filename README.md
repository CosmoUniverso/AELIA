# AEGIS Embryo

Prototipo v0.2 di organismo digitale 2D con corpo emergente, mondo simulato e controller cognitivo episodico.

## Avvio

```bash
pip install -r requirements.txt
python run_visual.py
```

## Controlli

- `SPACE`: pausa
- `R`: reset
- `S`: snapshot
- Click sinistro: inietta nutrienti
- Click destro: inietta tossine
